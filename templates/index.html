ICARTABLE- Module d'authentification utilisateur

Documentation API

ICARTABLE
Microservice		 Ce microservice sert a l'authentification des utilisateurs du system icartable, il ne s'agit pas du module de gestions de compte utilisateur, ce microservice se concentre sur la gestion de connexion des utilisateur et leurs session d'utilisation. il communique avec les autres micro service du system icartable a travers son API GraphQL. la liste de fonctionnalite et le protocole de communication incluant les requette qu'il est possible de faire et la facon de le faire est documente en bas. Thanks
  • Technologies utilisee :
  • Django & MySQL : Pour le backend et la base de donnees
  • graphene_django : Fourni l'API GraphQl pour dialoguer avec le module
  • graphene_jwt : Pour l'authentification des utilisateur
  • Fonctionnalites :
  • Creation d'un compte utilisateur
  • Activation du compte utilisateur via un code envoyer par email (obligatoire pour les professeurs)
  • Changement de mot de passe par l'utilisateur
  • Envoie des code de confirmations par email<
  • Suppression d'un compte utilisateur
  • Authentification d'un utilisateur par username et password
  • Generer un token d'acces identifiant l'utilisateur., permettant d'acceder aux autres modules securisee
  • Verifier la valide d'un token de connexion pour identifier les utilisateurs connectee
  • Rafraichir le token de connexion en fournissant le token de rafraichissement.

Procédure échelonnée pour réaligner votre rythme de sommeil

Jour Actions recommandées
Jours 1–3 Avancez le réveil de 15 min chaque matin ; exposez-vous à la lumière du jour, limitez café et évitez d’allonger la soirée devant les écrans.
Jours 4–6 Continuez à avancer le réveil ainsi que l’heure de coucher de 10 à 20 min par palier.
Poursuivez l’exposition lumineuse matinale, commencez une activité physique en journée, ajustez doucement la prise du café uniquement au matin.
Jours 7–10 Stabilisez le lever et le coucher ; concentrez-vous sur la constance et la rigueur, poursuivez toutes les bonnes habitudes d’alimentation, hydratation et sport matinal.
Jours 11–14 Placez le coucher entre 22 h et 23 h, pas de café après 13 h, dîner léger, environnement calme et rituel d’apaisement obligatoire en soirée.
Jour 15+ Maintenez durablement ce rythme : lumière du matin, hydratation, horaires synchronisés, repas adaptés et hygiène de sommeil stricte.

Exemple d’une journée idéale

  • 7 h : Réveil, pleine lumière naturelle (sortir, marcher, fenêtre ouverte…)
  • 8 h–8h30 : Petit-déjeuner + café du matin (jamais après 13 h)
  • 10 h–12 h : Activité physique (marche, sport modéré)
  • Déjeuner : Plats riches fibres/magnésium/potassium (légumineuses, bananes, légumes, oléagineux)
  • Après-midi : Hydratation continue, zéro caféine, snacks sains (fruits, graines, noix)
  • 19–20 h : Dîner léger, fini avant 21 h
  • 21 h : Écrans coupés, lumière tamisée, musique douce ou lecture
  • 22 h–23 h : Rituel apaisant puis coucher
Patience : le réalignement prend 2 à 3 semaines. Constance et rigueur sont les clés pour retrouver un sommeil réparateur.
